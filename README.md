# ポートフォリオ紹介

http://meigen.link/

このアプリは、ボタンをクリックするたびに偉人の名言を閲覧することができます。
辛い時や悲しい時、モチベーションをあげたい時に、偉人の名言に触れることで少しでもポジティブな気分になったり、励みになって欲しい、という想いで開発しました。

## 使用技術
*　フロントエンド
  * HTML/CSS
  * JavaScript
  * React
* バックエンド
  * Python
  * Frask（フレームワーク）
* インフラ・開発環境
  * AWS(EC2,VPC,S3,Route53)
  * Nginx
* 外部APIサイト
  * https://meigen.doodlenote.net/
